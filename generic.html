Project Description:

I created a digital etch a sketch using Java objects, interfaces, and nested classes. This project asks the user for the height and width of the board they will be drawing on. It then allows the user to choose which direction they want to move the stylus. After the stylus is moved, it asks the user for the next direction. This is repeated until the user completes the drawing.

The Board class displays the sketches. Boardable elements are placed in a cell on the board and the board keeps track of where it is. The Board is updated and displayed on creation and everytime a boardable element is moved. Cell is a nested class in Board. It creates the cells of the Board grid. If an element in the cell is visible, then the cell will become visible. If it is not visible, it will return '#'. If the cell is empty and visible it should return ' '. A cell that is visible and contains an element should return the element. The stylus is used to draw. It can move along the board passing a Direction. This takes input from the user to determine the direction.
